# OpenGL study

## mac 初始化

1. 安装 homebrew
2. 安装 glfw

```
brew install glfw
```

3. 安装 glad

```
brew install glad
```

## 配置

1. 已经配置好vscode的task，通过command + shift + b快捷键生成app可执行文件,
    再运行

    ``` bash
     ./app 
    ```
